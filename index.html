    ## Cleanup

  • A command line utility that organises files in a directory into subdirectories based on the file extensions.
  • Step 1: Go to the folder location which you want to organize.
  • Step 2 : Open your terminal
  • Step 3: Run Command:
    • $cleanup
  • Step 4: A new folder will be created with the name organized_files.
  • Step 5: Open this folder, and see everything is organised :)

    • ## Code Companion

  • Automation script which download test cases of coding problems from the platform and adds template according to your coding language.
  • You can save your time during contest time. You don't have to write the testcases again and again.
  • supported coding platforms: Codechef, Codeforces, Leetcode, Atcoder
  • give your coding platform (codeforces/codechef/leetcode/atcoder)
  • for leetcode, you need to give your leetcode credentials.
  • give your contest id. (you will find it on your contest URL. the id written at the end of the url)

  • choose your coding language (cpp/java/python).
  • now, script will will visit all the problems of the contest and download their testcases.
    • It will create multiple directories A B C D E inside the Problemset directory depending on the number of problems in contest. directory flow will look like this: code/codeforces/1492/A
    • each directory created will have
      • input.txt : containing problem inputs.
      • output.txt : containing problem outputs.
    $npm run getTestCases

    • ## Spotify Playlist Manager

  • Automation script which can manage your spotify playlist data.
  • Managing playlists with huge amount of data consumes a lot of time. With the help of this application you can manage your playlists fastly. You don't have to create playlists, add or delete songs manually again and again.
  • give your spotify credentials and choose your operation.
  • operations:
    • Create Playlist : Add songs to your new Playlist.
    • Modify any Playlist : Add/Remove songs from that Playlist.
    $npm run spotify-manage

    ## Youtube Playlist Analyzer

  • Sometimes you want to track down the duration of a youtube playlist. This Automation script will help you to calculate the following details of a playlist.
    • no of videos of the playlist.
    • length of the playlist.
    • length of playlist when covered in different speed formats : 1.25x, 5x, 1.75x and 2x.
    • playlist content in tabular form.
  • Step 1 : Open your terminal
  • Step 2 : Run below command and paste the youtube playlist link (for eg: https://www.youtube.com/playlist?list=PLf7o2Xlqx5OHfAX0mN4CDryeNIWDn3C3_)
    • $ytpAnalyzer
  • Step 3 : Done!
